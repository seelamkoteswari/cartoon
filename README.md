
# 🎨 Cartoonify App

A full-stack web application that converts images into cartoon-style avatars using OpenCV. It includes user authentication, multiple image filters, real-time camera processing, and image history with download functionality.

## 🚀 Tech Stack

Flask • OpenCV • Bootstrap • SQLite

## ✨ Features

* Image upload & cartoon filters
* Real-time webcam effects
* User login & registration
* Image history & download

## ▶️ Run

```bash
python run.py
```
